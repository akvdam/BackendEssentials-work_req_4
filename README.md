# BackendEssentials-work_req_4
Work requirement 4 for Backend Essentials PRO1002 course. 


## Project overview
This repository includes relevant files for the fourth work requirement in Backend Essentials. The main files include one script for the creation of the tables and the set up, one SQL file for the queries and one SQLite database. 


## Project structure
- read me file 
- git.ignore file
- a license file
- SQL file with the creation of the tables
- SQL file with the queries for the task
- a databasefile


## How to run the program
Make sure to have Python installed. 


#### By using the terminal and cloning the repository
Open the terminal, choose your desired folder and write the commmand:

```git clone https://github.com/akvdam/BackendEssentials-work_req_4.git```

#### By using the terminal and being in the project folder
Open terminal, access the project folder and run: 

```sqlite3 books.db < queries.sql```
